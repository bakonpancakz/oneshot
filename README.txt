---------------------------------------------------------------------------
DIRECTORIES
---------------------------------------------------------------------------

game
    This directory contains assets for the KUMA Game Project.

package
    This directory contains the source code for the YURI Toolkit, a custom
    archive format used exclusively by the OneShot Game Engine.

engine
    This directory contains the source code for the OneShot Game Engine.
    A cross-platform game engine designed for Windows 10/11, Linux Wayland,
    Nintendo Switch, Android 10+, and iOS/MacOS via Vulkan compatibility 
    layers.
